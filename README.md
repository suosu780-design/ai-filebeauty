# 科技委 · 报告解读（Word 转蓝主题长图）

在浏览器中上传含图文的 `.docx`，自动套用「报告解读」版式（蓝主题、落款「科技委 | 报告解读」、底栏固定二维码），并导出 **高清 PNG**（默认 3× 像素倍率）。

## 运行

```bash
cd ai_filebeauty
npm install
npm run dev
```

浏览器打开终端里提示的本地地址（一般为 `http://localhost:5173`）。

## 使用

1. 点击 **上传 Word**，选择 `.docx`。
2. 按需修改标题、署名、标签、数据来源、底栏文案与 **版心宽度**（720 / 1080 / 1440 / 1920 px）。上传后若检测到较宽或偏横版的插图，会自动建议 **1440px** 版心，可自行改回或加宽。
3. 点击 **导出高清 PNG**，将下载整页长图。

## Word 建议

- 首段或「标题 1」作为大标题；紧随其后的短段落可作为署名/日期行。
- 正文内大图会随版心宽度缩放，**不会**被压成固定小缩略图；宽图请优先选 **1440** 或 **1920** 版心以保证可读性。
- 复杂排版可能略有样式损失，重要内容尽量用 Word 内置标题样式。

## 二维码资源

底栏使用 `public/qr-footer.png`。若缺失，请将你的二维码 PNG 命名为 `qr-footer.png` 放入 `public/` 目录。

## 构建

```bash
npm run build
npm run preview
```

## 说明

- 极宽、极长的单页导出可能占用较多内存；若导出失败，可尝试减小版心宽度或缩短文档。
